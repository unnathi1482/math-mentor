# Algebra Formulas and Methods

## Quadratic Equations

### Standard Form
A quadratic equation has the form: ax² + bx + c = 0

### Quadratic Formula
For ax² + bx + c = 0, the solutions are:
x = (-b ± √(b² - 4ac)) / 2a

### Discriminant
The discriminant D = b² - 4ac determines the nature of roots:
- D > 0: Two distinct real roots
- D = 0: One repeated real root
- D < 0: Two complex conjugate roots

### Factoring Method
If ax² + bx + c = a(x - r₁)(x - r₂), then r₁ and r₂ are the roots.

### Sum and Product of Roots
For ax² + bx + c = 0 with roots α and β:
- Sum: α + β = -b/a
- Product: αβ = c/a

## Polynomial Identities

### Square of Binomial
- (a + b)² = a² + 2ab + b²
- (a - b)² = a² - 2ab + b²

### Difference of Squares
- a² - b² = (a + b)(a - b)

### Cube of Binomial
- (a + b)³ = a³ + 3a²b + 3ab² + b³
- (a - b)³ = a³ - 3a²b + 3ab² - b³

### Sum and Difference of Cubes
- a³ + b³ = (a + b)(a² - ab + b²)
- a³ - b³ = (a - b)(a² + ab + b²)

## Arithmetic Progression (AP)

### nth Term
aₙ = a + (n - 1)d
Where: a = first term, d = common difference

### Sum of n Terms
Sₙ = n/2 × [2a + (n - 1)d]
Or: Sₙ = n/2 × (first term + last term)

---

## Geometric Progression (GP)

### nth Term
aₙ = a × r^(n-1)
Where: a = first term, r = common ratio

### Sum of n Terms
Sₙ = a(rⁿ - 1)/(r - 1) for r ≠ 1

### Sum of Infinite GP (|r| < 1)
S∞ = a/(1 - r)

## Logarithms

### Definition
If aˣ = N, then x = logₐN

### Properties
- logₐ(MN) = logₐM + logₐN
- logₐ(M/N) = logₐM - logₐN
- logₐ(Mⁿ) = n × logₐM
- logₐa = 1
- logₐ1 = 0

### Change of Base
logₐb = logₓb / logₓa

## Inequalities

### Properties
- If a > b, then a + c > b + c
- If a > b and c > 0, then ac > bc
- If a > b and c < 0, then ac < bc (inequality flips)

### Absolute Value
- |x| < a means -a < x < a
- |x| > a means x < -a or x > a
